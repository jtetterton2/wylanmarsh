# Hello World Landing Page — Design Spec

**Date:** 2026-04-28

## Goal

A single bare HTML file that displays "Hello World" for use in a quick demo.

## Output

- `index.html` at the project root

## Structure

Standard HTML5 boilerplate with a single `<h1>Hello World</h1>` in the body. No CSS, no JavaScript, no external dependencies. Opened directly in a browser — no server required.

## Non-goals

- Styling or visual polish
- Responsive design
- Any additional content or interactivity
